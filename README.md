# NetKI

NetKI: Spectral graph representation in nearly linear time
required packages:

numpy
networkx
julia
scipy.sparse

to connect with julia, install the following packages
install julia interpreter: https://julialang.org/downloads/
open julia and type
using Pkg
Pkg.add("Laplacians")
Pkg.add("PyCall")
Pkg.add("IJulia")
Pkg.add("Distributions")
Pkg.build("PyCall")
